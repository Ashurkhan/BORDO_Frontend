## 🧪 Тестирование

### Тестовый сценарий

1. **Регистрация**
   - Перейти на `/register`
   - Заполнить форму
   - Отправить → должен произойти редирект на главную

2. **Просмотр объявлений**
   - На главной должны отображаться объявления
   - Фильтры должны работать
   - Клик на объявление → детали

3. **Создание объявления**
   - Перейти в профиль
   - "Создать объявление"
   - Заполнить форму → создание

4. **Избранное**
   - На странице объявления → "Добавить в избранное"
   - Перейти в "Избранное"
   - Объявление должно там быть

5. **Выход**
   - Профиль → "Выход"
   - Должен произойти редирект на логин

## 📞 Поддержка

### Если возникли проблемы:

1. **Frontend не загружается?**
   - Проверьте что npm run dev работает
   - Смотрите console в DevTools

2. **API ошибки?**
   - Проверьте что backend запущен
   - Смотрите network tab в DevTools
   - Прочитайте BACKEND_INTEGRATION.md

3. **Стили неправильные?**
   - Очистите браузер кэш
   - Перезагрузите страницу

4. **TypeScript ошибки?**
   - Запустите `npm run build`
   - Проверьте что типы импортируются правильно
