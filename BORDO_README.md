# 🐄 Bordo - Интернет-магазин скота

Современное React приложение для покупки и продажи скота онлайн с полным функционалом аутентификации и управления объявлениями.

## Функциональность

### Аутентификация
- ✅ Регистрация новых пользователей
- ✅ Вход в систему (логин)
- ✅ Сохранение токена в localStorage
- ✅ Автоматическая проверка сессии при загрузке

### Каталог объявлений
- ✅ Просмотр всех активных объявлений
- ✅ Фильтрация по категориям скота
- ✅ Поиск и сортировка
- ✅ Подробная страница каждого объявления
- ✅ Информация о продавце и контакты

### Управление объявлениями
- ✅ Создание нового объявления (для авторизованных пользователей)
- ✅ Редактирование собственных объявлений
- ✅ Удаление объявлений
- ✅ Просмотр статуса объявления (ACTIVE, SOLD, INACTIVE)

### Избранное
- ✅ Добавление/удаление объявлений в избранное
- ✅ Просмотр избранных объявлений
- ✅ Быстрый доступ к сохраненным объявлениям

### Профиль пользователя
- ✅ Просмотр личной информации
- ✅ История своих объявлений
- ✅ Управление профилем
- ✅ Выход из системы

## Технологический стек

- **Frontend**: React 19 + TypeScript
- **Маршрутизация**: React Router v6
- **HTTP клиент**: Axios
- **Сборка**: Vite
- **Стилизация**: CSS3 (CSS-in-file)
- **Backend**: Spring Boot (Java)

## Установка и запуск

### Предварительные требования
- Node.js 16+ и npm
- Spring Boot backend, запущенный на http://localhost:8080

### Шаги установки

1. Перейти в папку проекта:
```bash
cd college
```

2. Установить зависимости:
```bash
npm install
```

3. Создать файл `.env` (опционально, по умолчанию используется localhost:8080):
```bash
VITE_API_URL=http://localhost:8080/api
```

4. Запустить dev сервер:
```bash
npm run dev
```

5. Открыть в браузере: http://localhost:5173

### Продакшн сборка

```bash
npm run build
npm run preview
```

## API Endpoints

Приложение ожидает следующие endpoints на backend:

### Аутентификация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `GET /api/auth/me` - Текущий пользователь

### Объявления (Ads)
- `GET /api/ads` - Все объявления
- `GET /api/ads/{id}` - Детали объявления
- `POST /api/ads` - Создать объявление
- `PUT /api/ads/{id}` - Обновить объявление
- `DELETE /api/ads/{id}` - Удалить объявление
- `GET /api/ads/user/{userId}` - Объявления пользователя

### Категории
- `GET /api/categories` - Все категории
- `GET /api/categories/{id}` - Детали категории

### Локации
- `GET /api/locations` - Все локации
- `GET /api/locations/{id}` - Детали локации

### Избранное
- `POST /api/favorites/{adId}` - Добавить в избранное
- `DELETE /api/favorites/{adId}` - Удалить из избранного
- `GET /api/favorites/my` - Мои избранные объявления

## Использование

### Регистрация и вход
1. Перейти на страницу регистрации (/register)
2. Заполнить форму с данными пользователя
3. Пароль должен быть от 5 до 8 символов
4. После успешной регистрации будут автоматически залогинены

### Просмотр объявлений
1. На главной странице (/) отображаются все активные объявления
2. Можно фильтровать по категориям слева
3. Клик на объявление открывает его детали

### Создание объявления
1. Нужно быть залогиненным
2. Перейти в профиль → "Создать объявление"
3. Заполнить форму с информацией о скоте
4. Выбрать категорию и локацию
5. Нажать "Создать объявление"

### Добавление в избранное
1. Открыть любое объявление
2. Нажать кнопку "❤️ Добавить в избранное"
3. Просмотреть избранные объявления в разделе "Избранное"

### Профиль пользователя
1. Перейти в "Профиль" в навигации
2. Просмотреть личную информацию
3. Управлять своими объявлениями (редактировать/удалить)
4. Выход из системы

## Примечания

- Приложение использует localStorage для сохранения токена аутентификации
- Все запросы к API автоматически включают Bearer token
- Защищенные маршруты перенаправляют неавторизованных пользователей на /login
- Адаптивный дизайн для мобильных устройств
